Han pasado unas semanas desde mi último post, pero he estado algo ocupado.

Mañana comienza el periodo de codificación para el Google Summer of Code. Así
que, como dije anteriormente, he estado trabajando en Jangouts (haciendo algunas
contribuciones, arreglando algunos bugs, etc.). Pero también, la semana pasada
empecé a hacer algunos tests. No una suite de tests completa, solo tests para
algunos componentes; el código cambiará mucho con la migración a Angular 2,
hacer una suite de tests completa sería un trabajo extra inútil. Estos tests son
para aprender cómo está estructurado el código de Jangouts y saber cómo
funciona.

Dentro del periodo de community bonding, me mantuve en contacto con mis mentores
([@imobach](https://github.com/imobach) y
[@ancorgs](https://github.com/ancorgs)). Esta semana pasada tuvimos una llamada
de Jangouts (obviamente usamos Jangouts para reuniones) para hacer la
planificación para las próximas semanas. Tendremos un tablero de Trello para
mantener las tareas organizadas y reuniones diarias para seguir el progreso.
También, por mi parte, me comprometí a escribir un post en el blog cada semana
con un breve resumen del trabajo de la semana.

Los próximos días comenzará el proceso de migración a Angular 2, pero primero
necesito un un plan.
